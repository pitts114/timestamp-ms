API: Timestamp Microservice

A simple API that returns a date as a Unix timestamp and as a string in a natural language form.

Example Input:

https://calm-basin-13708.herokuapp.com/1450137600

https://calm-basin-13708.herokuapp.com/December 15 2015

Example Output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
